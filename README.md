# Student-Management-System
Student Management System is a full-stack web app built with React, Redux, Node.js, Express, MySQL, and JWT authentication. It lets users sign up, log in, and securely manage student records with add, view, update, delete, and search features through a simple, responsive dashboard for efficient academic data handling.

## Tech Stack

**Backend:**
- Node.js / Express.js
- MySQL (Workbench)
- JWT Authentication

**Frontend:**
- React
- Redux (State Management)
- React Router

## Project Structure

```
├── back-end/       # Express API server
└── front-end/      # React application
```

## Installation & Setup

### Backend
```bash
cd back-end
npm install
node index.js
```

### Frontend
```bash
cd front-end
npm install
npm start
```

## Features

- User Authentication (Login/SignUp)
- Student CRUD Operations
- Protected routes with JWT
- Dashboard with student management

## Environment Configuration

Add `.env` file in `back-end/` directory with your database and API configuration variables
